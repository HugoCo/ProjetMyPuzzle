L'archive contient : 

- un "Main.java" à exécuter
- une classe Fenetres, mère de toutes les classes fenêtres
- une classe Fenetre_Indices contenant la fenêtre fonctionnant quand l'utilisateur tente d'avoir des points pour les indices
- une classe FenetreAccueil permettant à l'utilisateur de choisir son pseudo (non fonctionnel), sa couleur de fond (non fonctionnel), son niveau de difficulté,
son fichier de Puzzle et de lancer la partie en cliquant sur Jouer.
- une classe Image_Indice pour les images des quizz
- une classe Piece permettant de definir des pièces de puzzle
- un dossier ImageINSA contenant les images des quiz
- des images : fondAccueil, fondEcran, Kaaris, logo utilisées dans le jeu

Le but est de faire le score le plus bas.

Pour mettre son image personnelle : Prendre une image en .jpg, la mettre dans le dossier ProjetMyPuzzle. La selectionner depuis le menu d'accueil "Selectionner un fichier".
Par défaut image.jpg dans le dossier
Choisir une image d'environ 650*350 pour que l'affichage soit possible
Attention le pseudo et la couleur de fond ne sont pas pris en compte.
Attention le bouton recommencer dans la fenêtre de jeu n'est pas fonctionnel
Attention dans cette version, La fenetre des indices n'est pas fonctionnelle.
